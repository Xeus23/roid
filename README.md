# Usage
<!-- usage -->
```sh-session
$ npm install -g roid
$ roid COMMAND
running command...
$ roid (-v|--version|version)
roid/0.0.0 darwin-x64 node-v12.16.1
$ roid --help [COMMAND]
USAGE
  $ roid COMMAND
...
```
